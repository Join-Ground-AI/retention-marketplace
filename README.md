# Ground Retention Skills for Ecommerce

13 AI-powered email & SMS retention workflows for DTC and ecommerce teams. Built for heads of lifecycle marketing, retention, and ecommerce.

## Install

**Step 1: Add the marketplace**
```bash
claude plugins marketplace add Join-Ground-AI/Ground-Retention-Skills
```

**Step 2: Install the plugin**
```bash
claude plugins install ground-retention@Ground-Retention-Skills
```

That's it. All 13 skills are now available — just describe what you need in plain English.

## What's Inside

### Flow Architecture & Audit
- **Flow Audit & Gap Finder** — Audits your email/SMS flows against a 15-flow benchmark map
- **Flow Builder & Drafter** — Builds complete Klaviyo-ready flows with triggers, timing, splits, and copy
- **Welcome Series Optimizer** — Splits your welcome by purchase status
- **Abandonment Flow Suite** — Coordinated cart, browse, and checkout recovery

### Segmentation & Personalization
- **Segmentation Auditor & Builder** — Builds RFM, engagement, lifecycle, and suppression segments
- **Customer Journey Mapper** — Finds where automations should exist but don't
- **Personalization Engine** — Dynamic content blocks and Klaviyo template code

### Performance & Optimization
- **Flow Performance Reviewer** — Monthly reviews benchmarked against Klaviyo 2026 data
- **Deliverability & List Health Auditor** — Authentication, reputation, and Gmail/Yahoo compliance
- **Winback & Churn Prevention Strategist** — 3-layer defense system

### Campaign Strategy
- **Campaign Calendar Planner** — Monthly calendar with BFCM strategy
- **SMS Strategy & Compliance** — TCPA compliance and 160-character copy
- **Retention Dashboard & Reporting** — Cohort analysis and executive reporting
